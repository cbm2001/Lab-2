The Bear Sting Game

Objective : To help the bear escape the bees !

Bear tries to avoid being stung by bees by running and jumping. The goal is to have as less stings as possible! The game gets over once the number of stings reaches 1000. Make the game more challenging by increasing the number and speed of the bees.
Good Luck!

Instructions

Press left, right, up, down keys to move the bear

You can adjust:
the bear's speed : px/ms
the number of bees
the bees' speed : px/ms
and the refresh period : ms
You can also add a bee to make it more challenging by clicking this button
